Оплата ChatGPT через Плати по миру

Выпуск виртуальной карты, пополнение через СБП и оплата подписки

Инструкция основана на документации «Плати по миру» по оплате ChatGPT картой «Для подписок». ChatGPT и API оплачиваются отдельно: для подписки ChatGPT используйте сайт ChatGPT и раздел Billing, а не billing в OpenAI Platform.

Этап 1. Подготовка перед оплатой

Карта, баланс, аккаунт ChatGPT и важные ограничения перед выбором тарифа.

Шаги 1–5
1

Открыть бота и мини-приложение

Перейдите в Telegram-бот @platipomiru_bot, нажмите Старт, затем Открыть приложение. Откроется личный кабинет «Плати по миру».

2

Выпустить карту для подписок

В мини-приложении выберите карту Для подписок, нажмите Выпустить карту и оплатите выпуск через СБП. После оплаты в кабинете появятся реквизиты карты.

3

Пополнить карту с запасом

Пополните карту рублями через СБП. Баланс карты «Для подписок» отображается в EUR, а оплата ChatGPT может списаться с конвертацией и комиссией сервиса, поэтому лучше держать небольшой запас сверх стоимости подписки.

4

Выйти из аккаунта ChatGPT

Перед началом оплаты выйдите из своего аккаунта ChatGPT. Это отдельный пункт в инструкции сервиса: дальше вход выполняется заново уже в нужном сценарии оплаты.

5

Проверить тариф и пробный период

По инструкции «Плати по миру», оплатить подписку с бесплатным пробным периодом этим способом обычно не получится. Для тарифа Go или Plus с бесплатным периодом используйте отдельные инструкции сервиса через PayPal или Apple ID.

Результат этапа: карта выпущена и пополнена, аккаунт ChatGPT готов к оплате, выбран подходящий сценарий: Plus/Pro или Go.

Этап 2. Оплата тарифов Plus и Pro

Основной сценарий из инструкции сервиса для оплаты Plus или Pro картой «Для подписок».

Шаги 6–13
6

Подключить VPN для Plus/Pro

По инструкции «Плати по миру», для Plus и Pro используйте VPN с локацией Новая Зеландия или Испания. После включения проверьте регион через 2ip.ru или аналогичный сервис.

7

Открыть инкогнито и войти в ChatGPT

Откройте приватную вкладку браузера, перейдите на chatgpt.com и войдите в свой аккаунт. В документации сервиса отдельно отмечено, что с почтой на домене .ru могут быть сложности.

8

Выбрать тариф Plus или Pro

Откройте выбор тарифа через профиль или кнопку обновления плана и выберите нужную подписку: Plus или Pro.

9

Ввести реквизиты карты

На экране оплаты укажите номер карты, срок действия и CVV из мини-приложения «Плати по миру».

10

Не сохранять способ оплаты

В инструкции сервиса отдельно указано: не сохраняйте способ оплаты. Если на форме оплаты есть такой переключатель или чекбокс, не включайте его.

11

Указать billing address

Заполните billing address в соответствии со страной выбранного VPN-региона. Используйте корректные данные и ориентируйтесь на подсказки ChatGPT, требования платёжной формы и поддержку «Плати по миру».

12

Подтвердить оплату в приложении

Проверьте тариф, сумму и период списания. Если всё верно, подтвердите оплату. При запросе подтверждения операции откройте мини-приложение «Плати по миру» и подтвердите платёж там.

13

Проверить активность подписки

После успешной оплаты откройте Settings → Billing в ChatGPT и проверьте активный план, историю платежей и управление подпиской.

Результат этапа: тариф Plus или Pro оплачен, подписка активна в ChatGPT.

Этап 3. Оплата тарифа Go

Отдельный сценарий из документации сервиса: для Go отличается VPN-регион и есть дополнительные рекомендации.

Шаги 14–21
14

Подключить VPN для Go

Для тарифа Go инструкция «Плати по миру» указывает VPN с локацией Франция. После включения также проверьте регион через 2ip.ru или аналогичный сервис.

15

Открыть инкогнито и войти в ChatGPT

Откройте приватную вкладку, перейдите на chatgpt.com и войдите в свой аккаунт.

16

Выбрать тариф Go

Откройте выбор подписки и выберите тариф Go. Если ChatGPT предлагает Go с бесплатным периодом, сервис рекомендует использовать отдельную инструкцию через PayPal или Apple ID.

17

Ввести данные карты

Введите реквизиты карты «Для подписок». В документации сервиса также указано, что иногда помогает иностранная фамилия в платёжной форме, если форма её запрашивает.

18

Не сохранять способ оплаты

Как и в сценарии Plus/Pro, не сохраняйте способ оплаты в форме ChatGPT.

19

Указать billing address для региона

Заполните billing address в соответствии со страной VPN-региона. Для Франции в документации сервиса приведён пример адреса, но перед вводом данных лучше свериться с актуальными подсказками сервиса и платёжной формы.

20

Подтвердить оплату

Подтвердите платёж через уведомление в мини-приложении «Плати по миру».

21

Проверить активность Go

После успешного списания проверьте, что тариф Go активен в ChatGPT.

Результат этапа: тариф Go оплачен или понятно, что нужен альтернативный сценарий оплаты.

Этап 4. Если платёж не проходит

Что проверить перед повторной попыткой и когда переходить на альтернативную инструкцию.

Проверка
22

Проверить базовые причины отказа

Проверьте баланс с учётом комиссии и конвертации, статус карты, правильность реквизитов, VPN-регион, billing address и выбранный тариф.

23

Сменить регион или устройство

В документации сервиса указано: если ошибка сохраняется, попробуйте другой VPN/регион, например Германия, Испания или США, а также оплату с другого устройства.

24

Использовать альтернативные инструкции

Для тарифа Go или Plus с бесплатным периодом сервис рекомендует отдельные варианты: оплата через PayPal или Apple ID. Ссылки на эти инструкции добавлены ниже.

25

Обратиться в поддержку

Если отказ повторяется, напишите в поддержку «Плати по миру» через бот и приложите текст ошибки или скрин оплаты.

Результат этапа: либо платёж повторно проходит, либо выбран альтернативный способ оплаты и есть данные для поддержки.

Установка OmniRoute и Claude Code

Полное руководство по настройке на Windows

Git

winget install --id Git.Git -e --source winget

Скачать с https://git-scm.com/install/windows

git --version

Node.js

Бесплатный провайдер от Alibaba Cloud с доступом к специализированным кодовым моделям.

Все эти провайдеры входят в состав OmniRoute и не требуют дополнительных API ключей от вас.

Настройка Claude Code

Файл конфигурации

Файл settings.json находится в папке:

C:\Users\ВАШ_ПОЛЬЗОВАТЕЛЬ\.claude\

Папка создается автоматически при установке Claude Code.

Добавить блок "env"

Откройте файл C:\Users\ВАШ_ПОЛЬЗОВАТЕЛЬ\.claude\settings.json и добавьте:

{ "env": { "ANTHROPIC_API_KEY": "ВАШ КЛЮЧ ИЗ OmniRoute", "ANTHROPIC_BASE_URL": "ВАША ССЫЛКА ИЗ OmniRoute, ПО УМОЛЧАНИЮ http://localhost:20128/v1", "API_TIMEOUT_MS": "3000000", "ANTHROPIC_DEFAULT_HAIKU_MODEL": "kr/claude-haiku-4.5", "ANTHROPIC_DEFAULT_SONNET_MODEL": "kr/claude-sonnet-4.5", "ANTHROPIC_DEFAULT_OPUS_MODEL": "qw/qwen3-coder-plus" } }
В полях ANTHROPIC_DEFAULT_HAIKU_MODEL, ANTHROPIC_DEFAULT_SONNET_MODEL и ANTHROPIC_DEFAULT_OPUS_MODEL вы сами задаёте модели, которые вам нужны

Подтверждение API ключа при первом запуске

После настройки файла конфигурации при первом запуске Claude Code может появиться вопрос:

Do you want to use this API key?

Необходимо подтвердить использование ключа:

1. Yes

Это действие подтверждает использование API ключа из настроек и позволяет Claude Code корректно подключиться к OmniRoute.

Возможные проблемы и решения

OmniRoute не запускается

Решение 1: Проверить версию Node.js (рекомендуется 22.22.2)

node -v

Решение 2: Переустановить OmniRoute

npm uninstall -g omniroute && npm install -g omniroute

Решение 3: Запустить с явным указанием порта

omniroute --port 20128

Ошибка "Tools for Native Modules"

Решение:

1

Запустить PowerShell от администратора

2

Выполнить команду

npm install --global windows-build-tools
3

Дождаться полной установки

Может занять 5-10 минут

Порт 20128 занят

Решение 1: Проверить кто занимает порт

netstat -ano | findstr :20128

Решение 2: Запустить OmniRoute на другом порту

omniroute --port 20129

Решение 3: Обновить ANTHROPIC_BASE_URL в settings.json

"ANTHROPIC_BASE_URL": "http://localhost:20129/v1"

Claude Code не видит OmniRoute

Проверка 1: Убедиться что OmniRoute запущен и работает

Проверка 2: Проверить доступность в браузере

http://localhost:20128

Проверка 3: Проверить правильность API ключа в settings.json

Проверка 4: Выполнить диагностику в Claude Code

/doctor

Важные примечания

OmniRoute должен быть запущен перед использованием Claude Code
По умолчанию OmniRoute работает на порту 20128
Рекомендуемая версия Node.js: 22.22.2 (наиболее стабильная для работы с OmniRoute)
Все модели доступны через единый эндпоинт http://localhost:20128/v1
